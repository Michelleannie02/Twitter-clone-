Tweeter
====

Twitterのクローンアプリを自作してみました。

## Description
・　主な機能：　

○Auth機能

○ツイート投稿

○いいね機能

○フォロー、フォロワー機能

○アンフォロー機能

○フォーロー中のユーザーのみのツイート表示

○ユーザー検索機能



・ DATABASE:　Real time database



・ Architeture: MVVC
